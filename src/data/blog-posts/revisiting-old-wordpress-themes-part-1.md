## Step 1: Set a baseline

- Check how many lines of code there are
- Is the site up to date?
- How many plugins does the site use?
- What is the performance like?

## Step 2: Initial Clean Up

- Unified spacing format in all files to tabs
- General code clean up
  - fix random indentation issues
  - remove comments and commented out code
  - delete unused files
  - consolidate small include files
